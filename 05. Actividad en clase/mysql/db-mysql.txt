1. Primero verifica que estás en la base de datos correcta:
sql
SHOW DATABASES;
2. Si estás dentro de la base de datos ESCUELA, sal primero:
USE mysql;  -- o cualquier otra base de datos
3. Luego elimina la base de datos:
DROP DATABASE ESCUELA;
4. Verifica que se eliminó:
SHOW DATABASES;
⚠️ ADVERTENCIA IMPORTANTE:
El comando DROP DATABASE es irreversible

Eliminará permanentemente toda la base de datos y todas sus tablas

Se perderán todos los datos definitivamente

--Si solo quieres eliminar una tabla específica (no toda la base de datos):

-- Primero selecciona la base de datos
USE ESCUELA;

-- Luego elimina una tabla específica
DROP TABLE nombre_de_la_tabla;
Ejemplo de eliminar tablas individuales:
USE ESCUELA;
DROP TABLE MATRICULA;
DROP TABLE GRUPO;
DROP TABLE ALUMNO;
